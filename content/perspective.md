# Perspective — 2026-04-16

How we see the world right now. Updated quarterly.

1. **Channel indifference is correct.** B2C, B2B, B2A — the value question is the same. Optimizing for one over the others is a fashion bet, not a value bet.

2. **AI orgs are an unbounded vehicle.** The company itself is the asset that compounds, not any single product inside it.

3. **Distribution arbitrage is closing fast.** Channels that were quiet 18 months ago are now crowded. Defensibility moves toward what the system *learns*, not where it ships.

4. **The cohort window is narrow.** For maybe a few more years, a small team can imprint a productive will on a market that is still under-defined. After that, the structure will be cast.

5. **Most "AI strategy" is theater.** Reading the right model release notes is not strategy. Building an org whose execution layer edits itself is.
