# Operating

How the company runs day-to-day.

## Mechanics

- **AI executes.** Tasks are picked up and finished by agents.
- **Humans seed.** Will, structure, taste — these come from humans, once. Maintenance is not a human job.
- **Every gap → a rule.** When an agent gets stuck or makes the wrong call, the missing context becomes a rule file in this repo.
- **Every patch → a skill.** When a human steps in to fix something, the fix is captured as a skill so the agent does not need the same human next time.
- **Truth → files.** Decisions made in chat are not real until they are written into a file.

## Cadence

- **Continuous:** agents run, generate work, log results.
- **Weekly:** review which assumptions held and which broke. Update files.
- **Quarterly:** update [perspective](./perspective.md). Replace what is no longer true.
