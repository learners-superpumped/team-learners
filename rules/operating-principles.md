# Operating Principles

Stated as invariants. The system is correct when each holds; broken when any fails.

1. AI executes. Humans do not maintain.
2. Humans seed will and structure. That is the only role.
3. Every gap an agent hits becomes a rule, written into this repository.
4. Every patch a human applies becomes a skill, written into this repository.
5. Truth lives in files, not in chat. Chat that matters becomes a file.
6. Default to the smallest viable thing. Add only when value is observed.
7. Weekly: review which assumptions held and which broke. Update the files.
