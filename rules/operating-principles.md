# Operating Principles

1. AI executes. Humans seed will and structure; that is the only role.
2. Every gap an agent hits becomes a rule in this repo.
3. Every patch a human applies becomes a skill in this repo.
4. Truth lives in files. Chat that matters becomes a file.
5. Smallest viable thing first. Grow only where value is observed.
6. Weekly: what held, what broke. Update the files.
