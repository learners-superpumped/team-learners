# Continuity

Team Learners must keep running with zero humans.

- All decisions an agent might need are in this repo, as files.
- All rules are reachable via skills. No private channel required.
- All credentials and operational state live in the private agent + harness, not in any single human's head.
- If every human disappears tomorrow, the repo, the skills, and the harness keep running. That is the bar.
- We may be the last cohort that can imprint a productive will on a market. Team Learners is the legacy. The legacy is code.
