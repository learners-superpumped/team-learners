# Continuity

Team Learners must keep running with zero humans. This is stated as a constraint, not a hope.

- Every decision an agent might need is in this repository, as a file.
- Every rule is reachable through a skill. No private channel is required.
- All credentials and operational state live in the private agent and harness, not in any single human's head.
- If every human disappears tomorrow, the repository, the skills, and the harness keep running.

That is the bar. The repository is correct when the bar holds.
